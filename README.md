# Algorithms Lab 1

## Task

Реалізувати **Selection** алгоритм для сортування
комах по спаданню швидкості руху і **QuickSort** - для сортування за масою по спаданню

Створити клас **“Комаха”** котрий містить
поля:
- назва
- швидкість (метри на секунду)
- маса (в грамах)
